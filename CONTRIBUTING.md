# Contributing to Tattoo Equipment Compatibility Checker

Thank you for your interest in contributing to the Tattoo Equipment Compatibility Checker! This tool is part of the Poli International Widget Suite, built for the body art community.

## 🎯 Ways to Contribute

### 1. Update the Database

The most valuable way to contribute is by adding new equipment or updating compatibility data:

- Machines (wireless, pen, rotary, coil)
- Power Supplies (traditional, batteries)
- Cartridge Systems (universal, brand-specific)
- Grips & Tubes

### 2. Report Bugs

Found a bug or incorrect compatibility info? Please open an issue with:

- Clear description of the problem
- Steps to reproduce
- Expected vs actual behavior

### 3. Suggest Features

Have an idea for a new feature? We'd love to hear it! Open an issue with:

- Clear description of the feature
- Why it would be useful

## 🛠️ Development Setup

1. **Fork and clone the repository**

   ```bash
   git clone https://github.com/Poli-International/equipment-compatibility-checker.git
   cd equipment-compatibility-checker
   ```

2. **No Build Process**
   This tool is built with pure HTML/CSS/JS. You can simply open `standalone-tools/equipment-compatibility/index.html` in your browser to test changes.

## 📋 Code Standards

- Maintain the mobile-first responsive design
- Use clean, commented JavaScript
- Follow the existing HSL color palette
- Keep the database files structured and easy to read

## 📝 Commit Messages

Use clear, descriptive commit messages:

- `feat: Add new FK Irons Flux Max models`
- `fix: Correct connection type for Peak Matrix Pen`
- `docs: Update installation instructions`
- `style: Improve mobile button spacing`

## 🤝 Community Guidelines

- Be respectful and inclusive
- Help others learn and grow
- Focus on constructive feedback
